Data.js

No magic, just a simple way to publish a string to one or multiple elements. Data.js is using the UMD pattern to make it work with AMD, CommonJS and as a global object on the window.

Install

This package is available on npm as mm-data. Install it using npm install mm-data or just download the mm-data.min.js.

Use it

Using Data.js is super easy, so to get you started look at the examples below. If you don't use an AMD or CommonJS system then you'll have mmData available as an object on window.

var Data = require('mm-data');

/** Broadcast a messag to all subscribers of "new_messages" */
Data('new_messages').broadcast('10');

/** Subscribe to "new_messages" */
Data('new_messages').subscribe(element);

/** Unsubscribe the element */
Data('new_messages').unsubscribe(element);

/** Unsubscribe all subscribers */
Data('new_messages').removeSubscribers();

/** You can chain methods any way you want */
Data('new_messages').subscribe(element).broadcast('20').unsubscribe(element);

/** You can also of course store the data instance in a variable */
var newMessages = Data('new_messages');

newMessages.broadcast('21');

When you subscribe an element, the element receives a data attribute, like this: data-mm-bind="new_messages". You can therefor set this attribute manually on elements in your HTML code. Usually you will use this method to subscribe elements. You can of course unsubscribe manually subscribed elements like you normally would.
