This value is listening to message: waiting

Type in mmData('message').update('Hi there!'); in the console to update the value.

You can remove all listeners or just on single elements. To remove a listener: mmData(element).unbind(). To remove all listeners: mmData('message').removeListeners();

You bind elements by adding the attribute data-mm-bind="message" to the element. If you want to programmatically bind an element you can use mmData(element).bind('message')
